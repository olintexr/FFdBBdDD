# FFdBBdDD - Fundamentos de Bases de Datos  - Por Olinto Rodríguez-Atencio
## Fundamentos de Bases de Datos — Resúmenes Didácticos

Este repositorio reúne resúmenes, explicaciones narrativas, ejemplos y material de apoyo para estudiantes que cursan **Fundamentos de Bases de Datos**, tomando como referencia el libro de Korth & Silberschatz.

El objetivo no es reemplazar el libro, sino **preparar al estudiante** para leerlo con claridad conceptual, entendiendo primero lo esencial y luego profundizando en el texto original.  
Aquí se construye un puente entre la teoría formal y la comprensión práctica.

---

## Propósito del proyecto

- Servir como guía introductoria clara y estructurada.  
- Presentar los conceptos fundamentales con un enfoque narrativo y didáctico.  
- Conectar cada capítulo con las secciones correspondientes del libro de Korth.  
- Proveer ejemplos SQL breves, comentados y con intención pedagógica.  
- Crear un repositorio modular, ordenado y escalable para uso académico.  
- Facilitar el estudio autónomo y la comprensión progresiva.

---

## Filosofía FFdBBdDD

1. Claridad antes que complejidad.  
2. Explicaciones narrativas que conectan conceptos.  
3. Aprendizaje progresivo, capítulo por capítulo.  
4. Conexión explícita con el libro de Korth.

## Política de contribuciones

Este repositorio es **solo de lectura para los estudiantes**.  
Todo el contenido es elaborado y mantenido por el profesor, con el fin de garantizar:

- Coherencia narrativa  
- Rigor técnico  
- Estructura uniforme  
- Calidad didáctica  
- Progresión ordenada de los capítulos  

Los estudiantes **no deben modificar, editar ni enviar cambios** al repositorio.  
Su función es **leer, estudiar y practicar** con el material disponible.

Cualquier actualización, corrección o ampliación será realizada exclusivamente por el mantenedor del proyecto.
